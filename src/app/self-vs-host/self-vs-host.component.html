The appParentDir provides dependency for app-child and appDir with @Host() decorator
If we remove appParentDir, app-child and appDir will not get dependency, even it is provided in App component
Also if we change @Host() on @Self() in app-child or appDir it will not get dependency But in case of HostListener it means the element directive is attached to
It will listen to the click events only on app-child component
