Self vs Host example

The appParentDir provides dependency for the app-child component and appDir directive with the @Host() decorator
If we remove appParentDir, app-child and appDir will not get dependency, even it is provided in the parent component's metadata
Also if we change @Host() on @Self() in app-child or appDir it will not get dependency But in case of HostListener it means the element directive is attached to
It will listen to the click events only on app-child component
